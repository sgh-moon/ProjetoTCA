Filetador de impressora 3D de garrafas pet

Resumo do projeto

Esse projeto tem como objetivo desenvolver uma máquina automatizada capaz de criar/transformar garrfas PET em filamentos de impressão 3D, tendo como principal abordagem a sustentabilidade para a criação dos mesmos."

Após diversas pesquisas, percebeu-se a alta demanda por filamentos de plásticos, em prol das preocupações com o meio ambiente e sustentabilidade da natureza, foi uma das motivações para o desenvolvimento desse projeto que diminuisse o impacto no meio ambiente e contribuísse de uma maneira positiva para a instituição que está financiando e apoiando o projeto que, seia o Centro Social Marista (Cesmar).

A filetadora é responsável por cortar a garrafa em filetes. Deu-se início com testes manuais para depois se pensar no desenvolvimento do filetador aumatizado. Até o momemento, ainda se está nos testes manuais, mas está sendo desenvolvido o sistema automatizado. Ate então com esses testes experimentais, percebeu-se um aproveitamento de quase 90% dos filamentos ainda em fase manual.

A máquina que, ainda está em fase de testes e ainda é um protóipo, produziu filamentos resistêntes e de notávelmente adequados para o objetivo principal. As conclusões tiradas a partir desse projeto, destacam a viabilidade para o desenvolvimento de filamentos para impressões 3D, já uqe, implementando no mercado de trabalho, podem gerar novos empregos, é um projeto de baixo nível de custos e ainda corrobora para a melhora do meio ambiente.

Abstract

This project aims to develop an automated machine capable of creating/transforming PET bottles into 3D printing filaments, with sustainability as the main approach for their creation

After several studies, it was noticed that there was a high demand for plastic filaments, in favor of concerns about the environment and sustainability of nature, which was one of the motivations for the development of this project that would reduce the impact on the environment and contribute in a positive way to the institution that is financing and supporting the project, which is the Centro Social Marista (Cesmar).

The threading machine is responsible for cutting the bottle into fillets. Manual tests were used to begin the process, and then the development of an automated threading machine was considered. At the moment, manual tests are still being carried out, but an automated system is being developed. So far, with these experimental tests, it has been observed that almost 90% of the filaments have been used in the manual phase.

The machine, which is still in the testing phase and is still a prototype, produced resistant filaments that are remarkably suitable for the main purpose. The conclusions drawn from this project highlight the viability of developing filaments for 3D printing, since, if implemented in the job market, they can generate new jobs, it is a low-cost project and also contributes to improving the environment.

O que é um filetador?

Filetadores de impressão, são materiais termoplásticos que são utilizados para imprimir objetos 3D por meio da deposição de material fundido por uma impressora 3D.

Para que serve os filetadores?

Além de servirem de ótimo material para o então resultado do projeto, terá uma utilização desse material o que não poluirá o solo, bueiros, rios, nascenes,etc.

O plástico é um dos materiais (se não o mais) que mais demora para entrar em decomposição. A poluição gerada por essas garrafas, prejudicam durante chuvas que, em decorrer da poluição, acabam ocasionando enchentes.

Como pretende-se arrecadar garrafas pet para o procedimento deste trabalho?

Como o Centro Social Marista (Cesmar), localiza-se em uma região de vulnerabilidade, existe pouquissímos locais em que há a coleta de lixo regular, por isso, tem se a ideia de pedir ajuda a comunidade para o arrecadação dessas garrafas para dar continuidade desse projeto futuramente. Como algo integrado para conseguir matéria-prima para as impressoras 3D.

Qual o impacto do descarte incorreto de garrafas pet no meio ambiente?

O descarte incorreto de garrafas pet tem um impacto significativo no meio ambiente, principalmente devido ao tempo de decomposição do plástico e à poluição que causa nos ecossistemas, e com base nesse problema pensamos em produzir filamentos de garrafa pet para impressora 3D a partir da reciclagem dessas garrafas plásticas. O impacto do filamento de garrafa pet, produzido a partir da reciclagem de garrafas plásticas, é principalmente positivo, com benefícios ambientais e econômicos. A reciclagem reduz a demanda por matéria-prima virgem e diminui o impacto da produção de plástico no meio ambiente, além de ser uma alternativa mais acessível para a impressão 3D.

O que pretende-se com esse trabalho?

Esse projeto tem como objetivo desenvolver um protótipo capaz de produzir filamentos para impressoras 3D com base em garrafas PET, o projeto visa integrar conceitos de sustentabilidade, reciclagem e tecnologias para um melhor aproveitamento dos plásticos, que seriam descartados de maneira inapropriada e incorreta na natureza, com o custo-benefício de produção dos filamentos, já que esses produzidos tendo o PET como matéria prima possuem um menor custo.

Prática: Como colocar em prática o projeto?

A máquina é um equipamento que recebe o material plástico em formato de filete, que passa por uma parte quente que irá aquecer até uma temperatura em que o plástico fique mole e passe por um outro sistema virando um filamento de impressora 3D.
